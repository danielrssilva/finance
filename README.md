### Prerequisitos

É necessário ter [Node.js](https://nodejs.org/en) ou [Yarn](https://yarnpkg.com) instalado

### Instalação

1. Clone o repositório
   ```sh
   git clone https://github.com/danielrssilva/finance
   ```
2. Instale os pacotes NPM/Yarn
   * npm
     ```sh
     npm i
     ```
   * yarn
     ```sh
     yarn i
     ```

3. Inicie a aplicação
   * npm
     ```sh
     npm start
     ```
   * yarn
     ```sh
     yarn start
     ```
